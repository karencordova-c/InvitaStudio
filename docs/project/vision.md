# Vision — InvitaStudio

## Visión del proyecto

InvitaStudio es una plataforma web enfocada en la solicitud y gestión de invitaciones digitales personalizadas.

El sistema busca reemplazar procesos informales realizados por redes sociales y mensajes privados, centralizando la gestión de pedidos, pagos y entregas en una sola plataforma.

---

## Problema que resuelve

Actualmente muchos servicios de invitaciones digitales trabajan de manera manual mediante:
- WhatsApp
- Facebook
- Instagram
- Mensajes directos

Esto genera:
- Pérdida de información.
- Confusión entre pedidos.
- Mala organización.
- Dificultad para dar seguimiento.
- Problemas de comunicación.
- Falta de profesionalización.

InvitaStudio busca solucionar estos problemas mediante una plataforma estructurada.

---

## Objetivo principal

Crear un sistema web profesional para:
- Registrar solicitudes.
- Gestionar pedidos.
- Organizar entregas.
- Centralizar comunicación.
- Mejorar la experiencia del cliente.

---

## Usuarios principales

### Cliente
Persona que solicita una invitación digital.

### Diseñador / Encargado
Persona encargada de elaborar las invitaciones.

### Administrador
Persona responsable del funcionamiento del sistema.

---

## Alcance V1

### Incluido
- Página pública.
- Catálogo de servicios.
- Formulario de solicitud.
- Gestión básica de pedidos.
- Validación manual de pagos.
- Panel administrativo.
- Consulta de estado.
- Entrega de archivos finales.

### NO incluido
- Editor visual de invitaciones.
- Login de clientes.
- Chat en tiempo real.
- Pagos automáticos reales.
- Automatización IA.
- Aplicación móvil.
- Multiusuario avanzado.
